### Структура модуля

```
telegram_bot.py
├── class MusicGrabberBot
│   ├── __init__() — настройка бота
│   ├── start() — запуск webhook/polling
│   ├── stop() — остановка бота
│   │
│   ├── # Обработчики команд
│   ├── cmd_start() — /start
│   ├── cmd_help() — /help
│   ├── cmd_settings() — /settings
│   ├── cmd_search() — /search
│   ├── cmd_queue() — /queue
│   ├── cmd_stats() — /stats
│   ├── cmd_history() — /history
│   │
│   ├── # Обработчики callback'ов
│   ├── cb_main_menu() — главное меню
│   ├── cb_settings_menu() — меню настроек
│   ├── cb_search_source() — выбор источника
│   ├── cb_search_quality() — выбор качества
│   ├── cb_download_track() — загрузка трека
│   ├── cb_preview_track() — предпросмотр
│   ├── cb_cancel_job() — отмена задания
│   ├── cb_retry_job() — повтор задания
│   │
│   ├── # Вспомогательные методы
│   ├── _check_access() — проверка доступа
│   ├── _get_user_settings() — настройки пользователя
│   ├── _format_result_message() — форматирование результатов
│   ├── _send_progress_update() — обновление прогресса
│   └── _cleanup_context() — очистка контекста
│
├── class UserContext
│   ├── chat_id
│   ├── search_query
│   ├── search_results
│   ├── current_page
│   ├── settings (dict)
│   └── created_at
│
└── class UserSettings
    ├── search_source (youtube/soundcloud/monochrome/all)
    ├── convert_to_flac (bool)
    ├── download_folder (str)
    ├── notifications_enabled (bool)
    └── min_bitrate (int)
```

### Хранение данных

**В памяти (runtime):**
- `user_contexts: dict[int, UserContext]` — контекст поиска
- `active_downloads: dict[int, list[str]]` — отслеживание загрузок

**В базе данных (новые таблицы):**
```sql
-- Настройки пользователей бота
CREATE TABLE telegram_users (
    chat_id INTEGER PRIMARY KEY,
    username TEXT,
    first_name TEXT,
    settings_json TEXT,  -- JSON с UserSettings
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    last_active TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- История действий бота
CREATE TABLE bot_logs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    chat_id INTEGER,
    action TEXT,  -- search/download/settings/etc
    details TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Связь jobs с chat_id
ALTER TABLE jobs ADD COLUMN telegram_chat_id INTEGER;
```
